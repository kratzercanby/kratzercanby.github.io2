Raspberry Pi Quick Setup

  • These are helpful instructions
  • Get and format a 4GB+ SD card to FAT32. If larger than 32GB, use SD Card Formatter 4.0 from SD Card Association website to remove any partitions. Then use FAT32 Formatter to format the large SD card to FAT32.
  • Download NOOBS from the RPi website and unzip them to the SD card.
  • Put the SD card in the RPi, plug in all the USB/HDMI devices, then plug in the power and get started.
  • Once the operating system is started, download Aaron's setup file. Open a terminal and copy and paste (with CRTL-SHIFT-v) the following: wget -O $HOME/Downlods/setup tinyurl.com/ack-rpisetup
  • Run setup from the Downloads folder and watch RPi get configured!
