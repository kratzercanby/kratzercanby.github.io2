Exporting Stata tables to LaTeX

  • Make sure the estout package is installed in Stata
  • Put eststo X: reg y x1 x2, robust to store the regression as X (can use any variabel name, like firstreg, or yreg).
  • Use esttab X using stata_1a.tex, replace to output stata_1a.tex with the X regression table
  • Import the table into LaTeX using \import{stata_1a.tex}
