Setting Up Git

  • First, check for a current SSH key on your computer.
  • If none exist, create one with ssh-keygen -t rsa -b 4096 -C "yourgitusername@github.com"
  • Press enter on "Enter a file in which to save the key".
  • Create a passphrase (better remember this).
  • Start the ssh-agent with eval $(ssh-agent -s)
  • Add SSH key to the ssh-agent using ssh-add ~/.ssh/id_rsa
  • Add the SSH key to your GitHub account
    • Copy the ssh key to your clipboard with (clip < ~/.ssh/id_rsa.pub) or (sudo apt-get -y install xclip && xclip -sel cli ~/.ssh/id_rsa.pub)
    • Go to GitHub settings page.
    • Go to SSH and GPG keys, and click New/Add SSH key.
    • For the title, name the computer specifically so you know which key belongs to which computer.
    • Paste the key into the "Key" field. Add SSH key.
